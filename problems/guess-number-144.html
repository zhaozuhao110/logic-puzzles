猜数问题

guess-number-144

教授在三个学生脑门上各贴了一张纸条，写着一个正整数，并告诉他们：某两个数的和等于第三个。每个学生能看到另外两个人的数，但看不到自己的。

教授依次发问：
问第一个学生：不能；问第二个：不能；问第三个：不能；
再问第一个：不能；再问第二个：不能；
第三个学生说：“我猜出来了，144！”

问：另外两个人的数是多少？
逻辑解析 / analysis

对“我可能是和/我可能是差”做迭代排除

步骤 1 / 4
1

对任意学生：看到 (x,y) 时，自己只可能是 x+y 或 |x-y|

因为三个数里必有一个是另外两个的和。对某个学生来说，他看见另外两人是 x 与 y，则自己的数只能是：x+y（自己是最大者）或 |x-y|（自己是最小者）。

2

第三个学生最终喊 144 ⇒ 他看到的两数之和为 144

第三个学生能在第二轮得出“我就是 144”，说明“我可能是和/我可能是差”里有一个最终被排除。
设他看到的两个数为 a 与 b（a<b），他最初的候选是 a+b 与 b-a。最终确定为 144，最典型的情况是：a+b=144，而 b-a 被对话排除。

3

用“别人为什么还不知道”去排除差值方案

取一组能触发二轮收敛的经典解：a=36、b=108。第三个学生看到 36 与 108 时，候选为：
a+b=144，b-a=72。

如果他其实是 72，那么第二个学生看到 (36,72) 后，自己的候选为 108 或 36。
但若他是 36，则三数将变成 (36,72,36)，有人会在第一轮就因为看见两个相同的数而立刻知道（与第一轮全员“不能”矛盾）。
所以在“第三个是 72”的世界里，第二个学生会在第二轮就确定自己是 108，不会继续回答“不能”。

4

因此第三个不可能是 72，只能是 144

现实里第二轮第二个学生仍回答“不能”，排除了“第三个是 72”的可能。
于是第三个学生锁定自己为 144，并反推出另外两人分别为：
36 和 108（36+108=144）。

洞见总结 / insight
  • 看到两数后，你的候选往往是和/差；对话用来逐轮排除。
  • “别人还不知道”意味着某些世界会导致别人更早知道，从而被排除。
  • 这类题经常出现“会导致两数相等”的分支，用它做反证非常有效。
