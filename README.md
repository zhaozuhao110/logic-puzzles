# Logic Puzzles | 经典逻辑思维挑战

一个专为逻辑爱好者设计的经典谜题集，包含博弈论、数学推理、编程思维等多个分类的挑战。

## 🌟 项目特色

- **经典汇集**：收录了海盗分金、称球问题、分割金条等家喻户晓的经典逻辑题。
- **极致视觉**：深邃的科技感配色，支持明暗模式无缝切换。
- **交互体验**：沉浸式的解题引导，详细的逻辑步骤拆解。
- **自动构建**：内置 Python 脚本，支持快速扩展和生成新的题目页面。
- **响应式设计**：完美适配手机、平板及电脑端。

## 🧩 已收录题目

- **称球问题**：三次称量找出次品球 (难度: 9/10)
- **猜牌问题**：通过逻辑对话排除不可能的组合 (难度: 8/10)
- **海盗分金**：经典的博弈论模型 (难度: 7/10)
- **燃绳问题**：不均匀绳子的计时艺术 (难度: 5/10)
- **分割金条**：利用二进制思维优化支付方案 (难度: 4/10)
- **喝汽水问题**：空瓶换购的数学计算 (难度: 3/10)
