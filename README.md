# Logic Archive | 逻辑题集

收录经典逻辑推理、数学思辨与博弈策略问题。以数字档案馆的严谨标准，重塑经典思维谜题的阅读与推演体验，助您随时进入深度的思考心流。

## 🌟 项目特色

- **高端档案馆质感**：深邃的奢华暗色调配色，搭配定制的金色弥散发光特效与高级毛玻璃质感，提供沉浸式的阅读体验，支持明暗模式无缝切换。
- **结构化知识图谱**：摒弃碎片化的堆砌，将经典题型系统性地拆解为不同的思维流派与难度阶梯（1-10级）。
- **沉浸式交互推演**：题目详情页包含“视觉演示 -> 交互推演 -> 关键步骤 -> 核心洞见”的完整沉浸式流程，带有平滑的步进器动画。
- **丰富的题库**：目前已收录 50 道精选逻辑谜题，涵盖逻辑、数学、博弈论、策略和编程五大分类。
- **自动化构建**：内置 Python 构建脚本，支持快速扩展和生成新的题目页面，自动同步更新 JSON 索引和分类视图。
- **纯粹的前端实现**：无框架依赖的纯 HTML/CSS/JS 实现，极致轻量，甚至支持 `file://` 协议下的完美离线本地浏览。

## 🧩 收录分类

- **逻辑推理 (Logic)**：信息差推导、真假话判断、状态排除（如：三个灯泡、盲人分袜、天堂帽子游戏）。
- **数学思辨 (Math)**：时间差建模、概率计算、极限推导（如：计算年龄、细菌分裂、时针重合）。
- **博弈论 (Game Theory)**：公共知识、利益分配、动态博弈（如：海盗分金、五囚犯、红眼狂犬病）。
- **策略规划 (Strategy)**：统筹优化、最少步骤、极限过河（如：称球问题、分割金条、燃绳问题）。
- **编程思维 (Programming)**：算法雏形、二进制思想、状态机（如：十二个球、舀酒难题）。

## 🛠 构建与扩展

如果你想添加新的逻辑题：

1. 使用 Trae IDE 的 `logic-analysis-page` 技能，通过自然语言生成符合本站风格的题目 HTML 文件。
2. 运行构建脚本更新索引：
   ```bash
   python3 scripts/build.py
   ```
3. 脚本会自动更新 `data/problems.json` 和 `data/problems.js`，新题目将立刻在首页和目录页生效。

## 🎨 视觉设计规范

本项目遵循“高端站”设计美学：
- **Typography**: Playfair Display (衬线大标题) + Noto Serif SC (宋体正文) 结合。
- **Color Palette**: 以深空黑 (`#050508`) 和金属质感面 (`#1a1712`) 为基底，辅以黄铜金 (`#d4a537` / `#f0c94d`) 作为核心强调色。
- **Animation**: 采用 150-300ms 的缓动曲线，卡片悬浮伴随微妙的 `translateY` 和内外阴影发光。
